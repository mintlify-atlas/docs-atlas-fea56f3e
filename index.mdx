The NVIDIA Open GPU Kernel Modules project provides the source code for NVIDIA's Linux kernel modules, enabling transparency, community contributions, and customization of the GPU driver stack. Starting with the Turing architecture, all four core kernel modules are available as open source under a dual MIT/GPLv2 license.

## Get up and running

<Steps>
  <Step title="Check your GPU">
    Verify your GPU is Turing (RTX 2000 series) or newer. See the [compatible GPUs list](/introduction#compatible-gpus).
  </Step>
  <Step title="Install prerequisites">
    Install kernel headers, GCC or Clang, and `make` for your Linux distribution.
  </Step>
  <Step title="Build the modules">
    ```bash
    make modules -j$(nproc)
    ```
  </Step>
  <Step title="Install the modules">
    Install the user-space driver components first, then install the open kernel modules:
    ```bash
    sudo make modules_install -j$(nproc)
    ```
  </Step>
</Steps>
